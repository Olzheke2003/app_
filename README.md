# Game Muster
Первая api-шка для тестовго задания

## Quick start

    git clone https://github.com/Olzheke2003/app_
## Виртуальное окружение
    python -m venv venv
    .\Scripts\activate
## Установка библотек
    pip install req.txt
## Миграции
    python manage.py makemigrations
    python manage.py migrate
## Run
    docker-compose up
